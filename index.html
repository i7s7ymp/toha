スキル

Go

Goのイディオムやベストプラクティスに沿った、読みやすく保守性の高いコードを書くことを心がけています。レイヤードアーキテクチャをはじめ、複数のアーキテクチャパターンでの実装経験があり、プロジェクトの規模や目的に応じた設計・開発が可能です。
Linux

Linux環境での開発経験があり、プロセス管理やネットワーク周りの挙動についても理解があります。straceやnetstat、tcpdumpなどのツールを活用しながら、プロセスやネットワーク機能に関するデバッグ対応も行ってきました。
AWS Solutions Architect

Vue.js

Docker

PostgreSQL

データベース操作に加えて、スロークエリの洗い出しや改善、インデックスの最適化など、パフォーマンスチューニングにも対応できます。
Oracle Database

Scrum

スクラムの原則である「透明性，検査，適応」に沿ってアジャイル開発を行うことができます。
GitHub
